# Amplify Fusion - Getting Started

Amplify Fusion Getting Started projects used in creation of the [Axway Amplify Fusion Demo Series on YouTube](https://www.youtube.com/playlist?list=PLSlCpG9zsECp6eGdEr4rH2Ig86b0g14U-). You can import the zip files into your tenant to help you get started with Amplify Fusion.

Project List
* GettingStarted - used for most of the Hello Fusion videos
* MyFusionMCPServer - used for the Hello Fusion MCP Server videos
* OpenAIConnector - used for the Hello OpenAI Connector videos
